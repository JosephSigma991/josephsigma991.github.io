Featured Project
IntelligenceOps
Decision Pack. Executive summary, decision impact, and operational routing from demo data

Problem: Airline operations generate decision-relevant data continuously. Most of it reaches decision makers too late, without clear attribution, or after the recovery window has already started closing.

Solution: IntelligenceOps is a flight operations intelligence system built around a frozen data contract, governed KPI logic, and decision-ready outputs. SQL Server ETL publishes validated analytical artifacts, then Python, Streamlit, Power BI, and ReportLab turn them into KPI views, RCA, delay-code intelligence, scenario simulation, and executive decision packs. GitHub Actions validates the contract before outputs are used.

Result: A reproducible workflow that turns operational data into governed, attributable, decision-ready output. Controllable, inherited, and reactionary delays are separated at code level, so accountability is routed before the meeting starts.
